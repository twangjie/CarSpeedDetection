# Car Speed Detection

A car speed detector using CNN for recognition and OpenCV for tracking.

![Alt Text](images/img1.png)

## Quick Start

### Prerequisites

- Python 3.6.x
- TensorFlow 0.8
- OpenCV 3
- PySDL2

### Running
Use NPM to start the server

```bash
python drawLine+NN.py
```
